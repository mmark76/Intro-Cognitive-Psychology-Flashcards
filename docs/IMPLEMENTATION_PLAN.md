# Implementation Plan

## Phase 1 - MVP

- Chapter map and summaries.
- 100 flashcards.
- Self-rating and local progress.
- Spaced-review queue.
- Quiz mode.
- Progress dashboard.
- Backup and restore.
- PWA installation and offline shell.

## Phase 2 - After verified use

Potential improvements are considered only after real use of the MVP:

- Search and filtering by key term.
- Written free-recall answers before reveal.
- Per-chapter mastery targets.
- Better quiz distractor generation.
- Study reminders.
- Optional import of revised content.

No backend, account system or cloud synchronisation is planned for the first phase.
