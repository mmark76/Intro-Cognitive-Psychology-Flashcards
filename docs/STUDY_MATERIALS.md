# Study Materials

The `Υλικό για μελέτη` page contains three built-in external resources and supports additional user-defined links.

## Adding material in the app

1. Enter the visible material name.
2. Paste an HTTP or HTTPS link from a cloud service or server.
3. Select `Προσθήκη link`.

Only the chosen name is displayed in the list. The full URL is used only as the link target.

Custom links are stored locally in the existing application settings and are therefore included in the normal JSON backup and restore flow.
