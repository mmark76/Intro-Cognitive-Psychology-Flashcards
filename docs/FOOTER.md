# Fixed Footer

The application footer is fixed to the bottom of the viewport and reserves enough page space so that it does not cover page content.

Displayed copyright notice:

`© 2026 Markellos Markides. All rights reserved.`

Legal navigation links:

- `License` → `#/legal/license`
- `Privacy` → `#/legal/privacy`
- `Analytics choices` → `#/legal/analytics`
- `Copyright protected` → `#/legal/copyright`

The legal pages are rendered inside the existing application layout and remain available through the hash router on GitHub Pages.
